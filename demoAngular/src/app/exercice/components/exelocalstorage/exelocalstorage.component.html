Exercice Local Storage

Dans votre project angular dans le module exos, ajouter des composants, un par produits cités plus bas, et un composant panier. Les composants de produits affichent le nom du produit, et un formulaire de quantité. Pour le formulaire de quantité :
- si le produit n'est pas présent dans le panier, utilisez un "select" comprenant les valeurs de 1 à 5, et un bouton "acheter".
- sinon, utiliser un "input" de type number avec déjà la quantité du panier, et un bouton "Mettre à jour".
Dans le composant panier, une liste des éléments sauvegardés doit être affiché : nom et quantité. Si pas de produits dans le panier, affichez un message "Panier vide".
Pour chaque produit du panier, afficher un bouton supprimant l'élément du panier. Les produits sont :
- Téléphone
- Ordinateur
- Batterie
